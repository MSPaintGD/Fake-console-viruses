# Fake Console Viruses
A collection of fake console “virus” scripts designed to simulate the look and behavior of malware in a command prompt or terminal — without causing any real damage.
These scripts are meant for fun, demonstrations, and learning, not for malicious use.

# About
Fake Console Viruses is a small project that contains scripts that imitate classic “virus-like” console behavior, such as:
Rapid scrolling text
Fake system warnings
Fake hacking messages
Infinite console spam
Random command outputs
Everything here is harmless and purely visual.
No files are deleted, modified, or stolen.
The goal is to replicate the look of old-school console viruses for entertainment or educational purposes
